# Test_technique_Clementine

## Comment démarrer le projet ?
### Backend 

Une fois le projet ouvert, acceder à la partie backend :
```bash
cd backend
```

Puis installer les dépendances :
```bash
npm install
```

Enfin démarrer la partie serveur :
```bash
npm run start:dev
```

## Techno utilisées

Node (NestJS), Vue3, Docker
